10.1: Word Equations

Key Concepts:

  • Chemical reactions convert reactants into products
  • Word equations describe reactions using names instead of formulas
  • General format: Reactants → Products
  • Word equations are useful for description but not for quantitative work

Examples:

Silver + Sulfur → Silver sulfide
Methane + Oxygen → Carbon dioxide + Water

10.2: Chemical Equations

Key Concepts:

  • Chemical equations use formulas instead of words
  • Skeleton equations show formulas with no relative amounts
  • Physical states: (s) solid, (l) liquid, (g) gas, (aq) aqueous
  • Reactants on left, products on right of yield arrow (→)

Example - Methane Combustion:

CH₄(g) + O₂(g) → CO₂(g) + H₂O(l)

10.3: Balancing Chemical Equations

Key Concepts:

  • Law of Conservation of Mass must be followed
  • Atoms cannot be created or destroyed in chemical reactions
  • Balance by adding coefficients, never change subscripts
  • Count atoms on both sides to verify balance

Steps for Balancing:

  1. Write correct formulas for all reactants and products
  2. Write the skeleton equation
  3. Count atoms of each element on both sides
  4. Add coefficients to balance each element

Example - Balancing Methane Formation:

Unbalanced: C + H₂ → CH₄
Balanced: C + 2H₂ → CH₄

10.4.1: Combination Reactions

Key Concepts:

  • Two or more reactants combine to form one product
  • General pattern: A + B → AB
  • Also called synthesis reactions

Examples:

2H₂ + O₂ → 2H₂O
4Na + O₂ → 2Na₂O
CaO + CO₂ → CaCO₃

10.4.2: Decomposition Reactions

Key Concepts:

  • One reactant breaks down into two or more products
  • General pattern: AB → A + B
  • Often requires heat or electricity
  • Opposite of combination reactions

Examples:

2H₂O → 2H₂ + O₂
CaCO₃ → CaO + CO₂
2KClO₃ → 2KCl + 3O₂

10.4.3: Combustion Reactions

Key Concepts:

  • Fuel reacts with oxygen gas
  • Products typically include CO₂ and H₂O
  • Releases heat and light energy
  • Hydrocarbons produce CO₂ and H₂O when burned completely

Examples:

CH₄ + 2O₂ → CO₂ + 2H₂O
C₃H₈ + 5O₂ → 3CO₂ + 4H₂O
2C₂H₆ + 7O₂ → 4CO₂ + 6H₂O
Note: Incomplete combustion can produce CO (carbon monoxide) instead of CO₂

10.4.4: Single Displacement/Replacement Reactions

Key Concepts:

  • One element replaces another in a compound
  • General pattern: A + BC → AC + B
  • Activity series predicts if reaction will occur
  • More reactive elements displace less reactive ones

Predicting Single Displacement Reactions:

  1. Check activity series to see if reaction occurs
  2. Identify which element is being replaced
  3. Write formulas with correct charge balance
  4. Balance the equation

Examples:

Zn + CuSO₄ → ZnSO₄ + Cu
2Al + 3CuCl₂ → 2AlCl₃ + 3Cu
Mg + 2HCl → MgCl₂ + H₂

10.4.5: Double Displacement/Replacement Reactions

Key Concepts:

  • Cations and anions exchange partners
  • General pattern: AB + CD → AD + CB
  • Common types: precipitation, acid-base, gas evolution
  • Products must have correct charge balance

Steps for Predicting Double Displacement:

  1. Switch the partners (cations with new anions)
  2. Correct formulas based on ion charges
  3. Balance the equation

Examples:

BaCl₂ + Li₂SO₄ → BaSO₄ + 2LiCl
AgNO₃ + NaCl → AgCl + NaNO₃
HCl + NaOH → NaCl + H₂O

Types of Double Displacement:

  • Precipitation: Forms a solid product (precipitate)
  • Acid-Base: Acid + Base → Salt + Water
  • Gas Evolution: Produces a gas product
